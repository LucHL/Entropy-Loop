# Entropy Loop

Welcome to Entropy Loop, a tactical strategy and card game! Play in an infinite world with a multitude of possibilities !!

## Minimal System Requirements

- **OS :** Windows 10/11
- **Processor :** Intel Core i3 or more
- **Memory :** 4GB RAM
- **Graphics :** Intel HD Graphics or equivalent
- **Storage :** 500 MB available space

## Patch Notes & Support
If you encounter a visual glitch or a bug in the game, a diagnostic file named `bugtracker.log` is automatically generated in the game directories. You can find the exact file path directly within the game Settings.

Feel free to share it with us to help improve the experience!
