# **Voidborn – Autochess Deck-Building Game**

Bienvenue dans *Voidborn*, un jeu tactique mêlant stratégie autochess et mécaniques de deck-building, le tout dans un univers sombre peuplé de créatures du Néant. Ce projet est développé en **Unity** avec une direction artistique **low-poly stylisée**.

---

## **Présentation du projet**

Dans *Voidborn*, le joueur constitue une équipe de combattants issus du Néant et les déploie sur un champ de bataille où chaque choix est crucial. Les combats sont automatisés (auto-battle), mais le placement, le choix des unités, et la construction du deck influencent fortement l’issue des parties.

---

## **Caractéristiques**

- **Genre** : Autochess, Deck-Building, Roguelite  
- **Plateformes cibles** : PC & Mobile  
- **Style graphique** : Low-poly isométrique  
- **Moteur de jeu** : Unity (URP)  
- **Langage** : C#  
- **Multiplateforme** : Oui (PC, Android dans le futur)  
- **Procédural** : Génération aléatoire d’arènes et d’ennemis  

---

## **Fonctionnalités principales**

- Système de combat autochess avec IA  
- Deck de cartes pour invoquer, booster ou altérer les unités  
- Progression roguelite entre les niveaux  
- Génération procédurale des arènes  
- Interface utilisateur intuitive compatible mobile  
- Optimisation URP pour performances fluides
